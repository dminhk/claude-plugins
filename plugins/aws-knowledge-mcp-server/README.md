# AWS Knowledge MCP Server

A fully managed remote MCP server providing up-to-date AWS documentation, code samples, API references, and regional availability data.

## Features

- Real-time access to AWS documentation, API references, and architectural guidance
- Regional availability data for AWS services and CloudFormation resources
- Best practices and getting started guides
- CDK and CloudFormation documentation
- Amplify framework materials for full-stack development

## Available Tools

| Tool | Description |
|------|-------------|
| `search_documentation` | Search AWS documentation with optional topic filtering |
| `read_documentation` | Convert AWS documentation to markdown format |
| `recommend` | Get content recommendations |
| `list_regions` | Retrieve AWS region information |
| `get_regional_availability` | Access regional data for services and resources |

## Requirements

- MCP client supporting HTTP transport
- Internet access
- No AWS account required
